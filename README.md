курсач по БД - web-портал авиакомпании на ADO.NET-ASP.NET-C#


WebPortal:

    DataBase - SkySharkDb.edmx - схема (с помощью Entity Framework) - нагенерены C# классы, байндинг DbContext, .sql запросы, инициализирующие базу, к ним дописаны метаданные для CRUD - в папке Metadata.
    
    Логика запросов\операций в папке BLL - всё в контексте C# LinqToEntity - слава ADO - не лезем руками в sql server.
    

    Вспомнил, нашёл я календарь нормальный тогда, ограничилось переводом на русский язык.


SeleniumTests:

    Пара пробных модульных тестов к бизнес-логике, заглушка контекста СУБД в FakeDbCOntainer.cs, несколько интеграционных "тестов"-скриптов для Selenium в /integral_tests
